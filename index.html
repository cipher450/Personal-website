TryHTML


TryHTML is a WYSIWYG web page builder with plenty of built-in HTML and CSS examples.

TryHTML has an extensive library of code including HTML and JavaScripts perfect for those who don't want, or know how to, write code. HTML has lists, links, images, forms, flash, video, audio, paragraphs, and the always difficult (at least for us) CSS examples. The windows are split into two with code to the left and preview to the right.

TryHTML can also convert images to Base64 and vice versa and even get the source code of a web page by entering the URL. You can also get any color in RGB with the color picker.

Whether you want to learn HTML, or just get it done, TryHTML is an excellent way to get started. As far as we know, it's also the smallest WYSIWYG editor out there.
